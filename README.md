### Requirements 
- MariaDB Installed 
- PowerBI Desktop Installed
- Data Source ODBC Installed and configured with your MariaDB


### Usage
- Run SQL Script on your database : CREATE TABLE INC.sql
- Open PowerBI File
- Get / Replace ODBC Data source
